# Desafio-DIO---Criando-um-Dashboard-de-Vendas-do-Xbox-com-Excel
------------------------------------------------------------------------

 Desafio da DIO para criação de um Dashboard no Excel com os dados extraidos da planilha par assinaturas do Xbox.

-------------------------------------------
Do que se trata o projeto?
-------------------------------------------
* Tem por finalidade utilizando a base de dados disponibilizado pela DIO Fazer todas as etapas de um ETL para estração e interpretação dos dados.
* Os dados são valores, pacotes, quando foi fechado, periodo que foi fechado, e extensões
* Por fim após feito todas as as etapas vou montar um DashBoard para melhor interpretação dos dados.
